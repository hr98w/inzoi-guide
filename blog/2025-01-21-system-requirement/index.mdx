# inZOI System Requirements

Planning to dive into the enchanting world of inZOI? Let's make sure your PC is ready for the adventure! Here's everything you need to know about the hardware requirements to run this immersive life simulation game smoothly.

![inZOI System Requirements](/blog/system-requirement.webp)

## Minimum System Requirements

To begin your journey in inZOI, your system should meet these basic requirements:

### Essential Hardware
- **Operating System**: Windows 10/11 (64-bit)
- **Processor**: Intel Core i5-10400 or AMD Ryzen 5 3600
- **Memory**: 12GB RAM
- **Graphics**: NVIDIA GeForce RTX 2060 (8GB) or AMD Radeon RX 5600 XT
- **DirectX**: Version 12
- **Network**: Broadband Internet connection
- **Storage**: 60GB available space
- **Additional Notes**: Requires a 64-bit processor and operating system

With these specifications, you'll be able to explore the world of inZOI at standard settings while maintaining stable performance.

## Recommended System Requirements

For the optimal inZOI experience with enhanced visual fidelity and smoother gameplay:

### Optimal Hardware
- **Operating System**: Windows 10/11 (64-bit)
- **Processor**: Intel Core i7-12700 or AMD Ryzen 7 5800
- **Memory**: 16GB RAM
- **Graphics**: NVIDIA GeForce RTX 3070 (8GB) or AMD Radeon RX 6800 XT
- **DirectX**: Version 12
- **Network**: Broadband Internet connection
- **Storage**: 75GB available space
- **Additional Notes**: Requires a 64-bit processor and operating system

## Additional Notes

- **Storage Type**: SSD storage is highly recommended for faster loading times
- **Graphics Drivers**: Latest graphics drivers are required
- **Internet Connection**: Stable broadband connection required for online features and updates
- **Display Resolution**: 1920 x 1080 (1080p) minimum recommended
- **Input Devices**: Keyboard and mouse required, controller support available

## Why These Requirements?

inZOI's detailed world and complex simulation systems demand robust hardware to deliver the intended experience. The recommended specifications ensure you can enjoy:

- Seamless world exploration with minimal loading times
- High-quality character models and fluid animations
- Advanced lighting, weather effects, and particle systems
- Smooth performance during complex simulations and events
- Enhanced draw distances and environmental detail
- Support for high refresh rate displays

Ready to start your adventure? Make sure your system meets these requirements, and you'll be all set to immerse yourself in the fascinating world of inZOI!

**Note**: Performance may vary based on system configuration. These specifications are subject to change during game development.
